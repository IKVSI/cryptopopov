# Task18: Нейросетевой протокол обмена ключами 8bit симуляция Евы

Реализация нейросетевого протокола обмена ключами с обучением двух сетей и замедленным обучением третьей
Ключ - это веса сетей.

# Описание протокола

    1) A и B независимо генерируют 2 сети
    2) A посылает на вход случайный вектор, генерирует результат
    3) A отправляет B результат и вектор
    4) B посылает на вход случайный вектор и генерирует результат
    5) B отправляет A результат
    6) Если результаты равны A и B поощряют сеть, не равны переходим к шагу 2
    7) A генерирует сообщение применив веса как ключ для шифра и отправляет B
    8) B пытается расшифровать своим ключом из весов сообщение
    9) Если B расшифровывает сообщение, то ключ синхронизирован, обмен завершён, если нет - переход к шагу 2

    10) Ева делает всё тоже самое, только поощряет сеть если все 3 результата совпали, тем самым с каждым сопадением
    у A и B и не совпадением у E, E начинает отставать от них

## Использование

Пользователю предлагается понаблюдать за деятельностью Евы.
Он также может изменить количество заходов и посмотреть статистику.

### Help программы:

usage: main.py [-h] [-n N]

Протокол обмена ключом 8bit через нейросеть, симуляция Евы

optional arguments:
  -h, --help  show this help message and exit
  -n N        Количество ключей для обмена (запусков)
